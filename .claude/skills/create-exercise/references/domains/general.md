# Domaine : Général (fallback)

## Types d'exercices

Tous les types sont disponibles. Choisir selon la nature du contenu :

| Contenu | Type recommandé |
|---------|----------------|
| Concept, définition, principe | `multiple_choice` |
| Propriété, idée reçue | `true_false` |
| Application pratique (code) | `code` |
| Application pratique (calcul formel) | `calculation` |

**Règle par défaut :** mix de `multiple_choice` et `true_false`, avec `code` si le chapitre contient du code, `calculation` si le chapitre contient des formules.

---

## Multiple choice : bonnes pratiques générales

- Question sur la compréhension, pas la mémorisation
- 4 options avec distracteurs plausibles (pas trivialement faux)
- Explication en 2-4 phrases, éducative

## True/False : bonnes pratiques générales

- Affirmation claire, non ambiguë
- Tester des concepts importants ou des idées reçues
- Explication avec contexte additionnel
